# Xbox Gamepad Control Utility

Эта утилита позволяет полноценно и плавно управлять ОС Windows с помощью геймпада Xbox через графический интерфейс.

## Особенности
* Удобный современный графический интерфейс для тонкой настройки (GUI).
* Статичное перемещение мыши для попиксельной точности (настройка чувствительности прямо в приложении).
* Полная поддержка Drag-and-Drop.
* Скроллинг и управление вкладками браузера.
* Работа с мультимедиа (громкость, след/пред трек, Play/Pause).
* Вызов встроенной экранной клавиатуры.
* Моментальное сохранение настроек - все изменения сразу записываются в `config.json`.
* Горячее переподключение (если геймпад "засыпает", утилита подхватит его сама).

## Установка и запуск
1. Убедитесь, что установлен Python 3.
2. Установите зависимости (если еще не сделали):
   ```cmd
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```cmd
   python main.py
   ```
   *Откроется окно с настройками.* Вы можете переименовать файл `main.py` в `main.pyw`, чтобы не открывалась фоновая черная консоль.

## Настройка через GUI
В открывшемся окне `Xbox Gamepad Utility` вам доступны ползунки:
* **Sensitivity**: Скорость перемещения курсора мыши.
* **Update Rate (Hz)**: Частота опроса геймпада (рекомендуется 60-120).
* **Deadzones (Мертвые зоны)**: Порог отклонения стиков/курков, до которого движение не регистрируется. 

Для изменения раскладки (Bindings) пролистайте интерфейс вниз. Радом с каждой кнопкой есть выпадающий список (ComboBox). Просто выберите новое действие. Все изменения сразу применяются.

## Запуск эмуляции
Нажмите большую зеленую кнопку **START** в правом верхнем углу интерфейса.
Статус поменяется на "Running".
Чтобы остановить эмуляцию (отвязать геймпад), нажмите красную кнопку **STOP**.

## Использование (По умолчанию)
- **Левый стик**: Движение курсора мыши.
- **Правый стик**: Скролл.
- **A**: Левая кнопка мыши (позволяет зажимать для выделения и перетаскивания).
- **B**: Правая кнопка мыши.
- **X**: Экранная клавиатура (osk.exe).
- **Y**: Enter.
- **LT / RT**: Масштабирование (сочетание Ctrl + колесико мыши).
- **LB / RB**: Назад / Вперед в браузере.
- **Крестовина (Вверх / Вниз)**: Громкость +/-.
- **Крестовина (Влево / Вправо)**: Предыдущий / Следующий трек.
- **Start**: Play/Pause медиа.
- **Back**: Свернуть все окна (Win+D).
- **Левый клик стиком (L3)**: Средняя кнопка мыши.
- **Правый клик стиком (R3)**: Закрыть вкладку (Ctrl+W).
